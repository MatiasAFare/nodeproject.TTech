# Pre Entrega proyecto NodeJS


### Script CLI para listar, crear y eliminar productos usando https://fakestoreapi.com

Comandos (desde la carpeta Proyecto):
- Listar todos: npm run start -- GET products
- Ver por id: npm run start -- GET products/15
- Crear: npm run start -- POST products "T-Shirt-Rex" 300 remeras
- Eliminar: npm run start -- DELETE products/7

O usar node directamente:
node index.js <METHOD> <path> [args]
